# Resume Builder + Interview Questions Generator

## 🚀 Project Description
This is a simple web tool that generates a resume based on user input and provides interview questions based on selected skills like HTML, CSS, JavaScript, Python, etc.

## 🌐 Live Project
https://resume-builder-interview-questions.vercel.app

## 💻 Features
- Resume Generator
- Interview Questions based on skills
- Simple and clean UI
- Beginner-friendly project

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 🎯 Purpose
I built this project to practice frontend development and understand how real-world tools work.
